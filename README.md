# RSS feed aggregator with post autoupdate feature

## Demo (deployed with Vercel): https://rss-feeds.vercel.app/


## Example:
![Example](/upload/example.gif)

## Installation

```bash
$ make install
```

## Run developer server

```bash
$ make develop
```

## Run linter

```bash
$ make lint
```

## Build project

```bash
$ make build
```
